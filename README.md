تطبيق ويندوز خفيف وذكي يقوم بتتبع البرامج المفتوحة اعتماداً على عنوان النافذة.  
موجّه للمطورين والطلاب، ويتيح تفعيل أو تعطيل تتبع كل برنامج بسهولة، مع دعم كامل لتتبع المتصفحات عبر كلمات مفتاحية متعددة.

## المزايا
- تتبع لحظي للنوافذ المفتوحة
- نظام ديناميكي يعتمد على Dictionary و LINQ
- دعم كامل للمتصفحات (Chrome, Edge, Firefox, Opera, Brave, Safari)
- إمكانية تخصيص البرامج المتتبعة عبر CheckBox
- تصميم نظيف وقابل للتوسّع
- مبني باستخدام C# و WinForms
